id — уникальный идентификатор поста в Product Hunt
slug — короткий URL-идентификатор (часть ссылки)
name — название продукта
url — ссылка на страницу продукта на Product Hunt
createdAt — дата и время публикации поста (ISO 8601)
featuredAt — дата и время, когда продукт попал в "featured" (главную); null если не зафичерен
tagline — короткий слоган/описание (одна фраза)
description — расширенное описание продукта
votesCount — количество голосов (лайков)
commentsCount — количество комментариев
topics — список тем (категорий), к которым относится продукт
thumbnail_url — миниатюра (иконка) продукта
website — внешняя ссылка на сайт продукта
media — список медиафайлов (скриншоты, видео); включает type, url и videoUrl
